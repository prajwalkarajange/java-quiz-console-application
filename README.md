# 🎯 Java Quiz Application

## 📖 Overview

A console-based Quiz Application developed using **Java** and **Object-Oriented Programming (OOP)** principles.

The application presents multiple-choice questions to the user, accepts answers through the command line, evaluates responses, calculates the final score, and provides feedback based on performance.

This project was built to strengthen my understanding of **Java fundamentals**, **OOP concepts**, **user input handling**, **arrays**, **loops**, and **application flow control**.

---

## ✨ Features

✅ Interactive command-line quiz

✅ Multiple-choice questions

✅ User answer collection using Scanner

✅ Automatic score calculation

✅ Performance feedback for perfect scores

✅ Object-Oriented Design

✅ Clean separation of classes and responsibilities

## 📂 Project Structure

```text
├── Main.java
├── Question.java
└── QuestionService.java
```

### 🚀 Main.java

Acts as the entry point of the application and controls quiz execution.

### ❓ Question.java

Represents a Question object containing:

* Question ID
* Question Text
* Four Options
* Correct Answer

### ⚙️ QuestionService.java

Handles:

* Quiz initialization
* User interaction
* Answer collection
* Score evaluation
* Result generation
